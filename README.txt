# Medication Tracker — PWA (GitHub Pages ready)

Upload these files to a public GitHub repository, then enable GitHub Pages to publish your installable web app.

## Steps (no command line)
1. Create a **public** repo on GitHub (e.g., `medication-tracker`) and tick **Add a README**.
2. In the repo, click **Add file → Upload files** and drag **all files from this folder** (not the folder itself).
3. Click **Commit changes**.

## Enable Pages
1. Go to **Settings → Pages**.
2. Set **Source** to `main` and **Folder** to `/ (root)` → **Save**.
3. Your app will be live at `https://<your-username>.github.io/<repo-name>/` within a minute.

## Install
- **Android (Chrome):** ⋮ → **Install app** (or **Add to Home screen**).
- **iPhone (Safari):** Share → **Add to Home Screen**.
